# Wetube 2022
 
### Global Router
- / -> Home
- /join ->  Join
- /login -> Login
- /search -> Search


### User Router
- /users/edit -> Edit user
- /users/delete -> Delete user


### Video Router
- /videos/watch -> Watch Video
- /videos/edit -> Edit Video
- /vieos/deletem -> Delete Video
- /videos/comments -> Comment on a video
- /video/comments/delete -> Delete a Comment of a video
