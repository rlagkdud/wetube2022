# Wetube 2022
 
### Global Router
- / -> Home
- /join ->  Join
- /login -> Login
- /search -> Search


### User Router
- /users/:id -> See user
- /users/logout -> Log out
- /users/edit -> Edit my profile
- /users/remove -> Delete my profile


### Video Router
- /videos/:id -> See Video
- /videos/:id/edit -> Edit Video
- /videos/:id/delete -> Delete Video
- /vidoes/upload -> Upload video
